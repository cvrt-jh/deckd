# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Email:** jh@cavort.de

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

I aim to respond within 48 hours and will work with you on a fix before
any public disclosure.

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
